# Postgresql_DB
Creating CoffeeShop DataBase in Postgres then export views &amp; materialized views from CoffeeShop DB and import to Mysql_DB and DB2

## Creating ERD in postgres by pgadmin ERD tool
- Customer
- Staff
- Produts
- Product_type
- Sales_transaction
- Sales_detail
- Sales_outlet
